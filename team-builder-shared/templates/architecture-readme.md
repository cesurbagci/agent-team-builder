# Mimari Dokümanlar

Bu dizin projenin mimari kararlarını, kısıtlarını ve tasarım notlarını barındırır.
Yazım/okuma standardı: `templates/doc-standard.md`. Buraya **sadece architect** yazar.

## İçindekiler
- `ilkeler.md` — cross-cutting prensipler.
- `adr/` — Mimari Karar Kayıtları (ADR). _(Şablon: `templates/adr.md`)_
- `constraints/` — mimari kısıtlar / guardrail'ler. _(Şablon: `templates/constraint.md`)_
- `design/` — tasarım notları / diyagramlar. _(Şablon: `templates/design.md`)_
